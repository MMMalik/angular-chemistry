scientificFilter

Info

A filter which applies the scientific notation to a number.

Usage

Param Type Details
value number|string An input value. It can be either a number or a string (it will be parsed internally to number). If it represents an invalid number, the input value will be returned unchanged. Moreover, the leading zero in fractions can be omitted, e.g. .34, .0022, and ".987" are considered as valid inputs.
precision number

An optional parameter. The input value will be displayed with the specified precision. By default, no rounding is performed. The max value for this param is 10.

symbol string

An optional parameter. The symbol between the significand and the exponent. The possible values are: "dot" for · and "cross" for ⨯. Default value is "dot";
