Info

The chemEntity directive allows you to render chemical formulas and reaction arrows. As a main feature, this directive parses the formula and inserts <sub></sub> and <sup></sup> where needed.

Usage

This directive creates an isolated scope. It can be used either as an element <chem-entity></chem-entity> or as an attribute <div chem-entity></div>. It has a default priority 0.

Param Type Details
chemFormula string

A string or {{ expression }} to eval. The formula can be given using simplified syntax, e.g. H2SO4 or Fe2(PO4)3. The parser places a number inside <sub></sub> tags if it is preceeded by a letter or a bracket (), ], or }). A similar approach can be taken to show ions, e.g. [Zn(NH3)4]^2+. The parser places the charge inside <sup></sup> tags if it is preceeded by ^ sign and is ended with - (hyphen) or +.

The simplified approach is not always appropriate. For example, it will fail to render &eta;^2-O2 correctly (see examples below).

A more explicit syntax can be used at any time. Everything placed inside _(here) will be wrapped with <sub></sub> tags, whereas content placed within ^(here), will be wrapped with <sup></sup> tags. For example, NH_(4)^(+) or Al_(2)Si_(4)O_(10)(OH)_(2). This way, &eta;^(2)-O2 will be rendered correctly. The explicit and simplified syntax can be mixed. e.g. NH4^(+) or C_12H_(22)O_(12).

The html entities can be used within formulas.

chemSub string

A string or {{ expression }} to eval. Will be passed to class attribute in each <sub> tag in the formula. Must be a defined css class, e.g. <style> .red{ color: red; } </style> (example below).

chemSup string

A string or {{ expression }} to eval. Will be passed to class attribute in each <sup> tag in the formula. Works analogously to chemSub (example below).

chemAsIs string

A string or {{ expression }} to eval. If set to "true", the formula will not be parsed (example below).

Examples

Output Code
<chem-entity chem-formula=""></chem-entity>
<chem-entity chem-sub="red" chem-formula=""></chem-entity>
<chem-entity chem-sup="red" chem-formula=""></chem-entity>
<chem-entity chem-as-is="true" chem-formula=""></chem-entity>
