chemScheme

Info

The chemScheme directive allows for rendering of chemical reactions. It serves as a wrapper for a set of chemEntity. chemScheme inserts + signs between reagents and an arrow between substrates and products.

Usage

This directive creates an isolated scope. It can be used either as an element or as an attribute: 

<chem-scheme ... ></chem-entity>
<ANY chem-scheme ... ></ANY>
It has a default priority 0.

Param Type Details
chemArrow string

A string or {{ expression }} to eval. A one-way binding. It wraps the expression with & and ;. Although it accepts any html entity, the reasonable values are:

  • for , .

chemSubst [string]

Accepts an Array of strings. A two-way binding.

chemProd [string]

Accepts an Array of strings. A two-way binding.
