chemScheme

Info

The chemScheme directive allows for rendering of chemical reactions. It serves as a wrapper for a set of chemEntity. chemScheme inserts + signs between reagents and an arrow between substrates and products.

Usage

This directive creates an isolated scope. It can be used either as an element <chem-scheme></chem-scheme> or as an attribute <div chem-scheme></div>. It has a default priority 0.

Param Type Details
chemArrow string

A string or {{ expression }} to eval. Possible values are:

  • for , .
