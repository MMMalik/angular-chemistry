tempFilter

Info

A filter for displaying a temperature value. It also enables converting between Celsius, Kelvin and Fahrenheit units. The filter does not check if the input value is below absolute zero. The result may therefore lack physical sense.

Usage

Param Type Details
value number|string An input value. It can be either a number or a string (it will be parsed internally to number). If it represents an invalid number, the input value will be returned unchanged. The leading zero in fractions can be omitted, e.g. .34, .0022, and ".987" are considered as valid inputs.
unit string

An optional parameter. The unit which will be displayed along with the value. Defaults to "K". The following values are supported for this parameter:

  • "K" for Kelvins, "C" for Celsius, "F" for Fahrenheit,
  • "K-to-C", "K-to-F", "F-to-C", "F-to-K", "C-to-K", "C-to-F" for converting.

precision number

An optional parameter. The input value will be displayed with the specified precision. Defaults to 1. The max value for this param is 10.

Examples

Try it:
Result:
Plain Result Code
<span ng-bind-html="(number | temp:'C')"></span>
<span ng-bind-html="(number | temp:'C-to-F')"></span>
<span ng-bind-html="(number | temp:'C-to-F':3)"></span>
